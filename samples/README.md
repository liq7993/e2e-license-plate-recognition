# samples/

把用于演示 / 自测的示例图片放在这里。

GUI 用法：

```bash
python zhongduan.py
# 在弹出的窗口里点击「加载图像」 → 选择 samples/ 下的图片
```

建议的命名：

```
samples/
├── car_blue_01.jpg     # 蓝牌示例
├── car_green_01.jpg    # 绿牌（新能源）示例
└── car_tilt_01.jpg     # 带倾斜/透视的示例，用于演示 STN 校正
```

> 出于版权与隐私考虑，本仓库默认 **不** 内置示例图片。请使用：
>
> - 你自己拍摄、并打码车牌敏感字段的照片；
> - 或 CCPD / BLPD 数据集中可公开演示的样本；
> - 或公开许可下的素材库。
>
> 不要把含真实车牌的非授权照片直接提交到本仓库。
